dwm Cheat Sheet
================================
###### Close Window
    [Shift]+[Alt]+[c]
###### Window Focus Left
    [Alt]+[j]
###### Window Focus Right
    [Alt]+[k]
###### Find/Run Program
    [Alt]+[p]
###### Toggle Master/Stack
    [Alt]+[Enter]
###### Switch Workspace (Tag)
    [Alt]+[1:9]
###### Send Window To Workspace (Tag)
    [Shift]+[Alt]+[1:9]
###### Quit Window Manager
    [Shift]+[Alt]+[q]
